Opis

Ten plugin wyświetla wizytówkę z serwisu Nasza-Klasa.pl. Wizytówki te są domyślnie wyłączone - aby ją włączyć, wejdź na stronę http://nasza-klasa.pl/profile/card/edit

English: This plugin shows user card from most popular Polish community site - Nasza Klasa (Our Class). It shows it as a widget.

Instalacja

  1. Wgraj katalog nasza-klasa-wizytowka do katalogu /wp-content/plugins/
  2. Włącz plugin poprzez menu 'Wtyczki' w WordPressie
  3. Ciesz się swoją wizytówką z NK na swoim blogu :)
